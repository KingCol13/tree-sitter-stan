===================
Operator Precedence
===================

a / b * c  = (a / b) * c

a + b + c = (a + b) + c

a * b + c = (a * b) + c

a + b * c = a + (b * c)

2 * x + 3 * -y = (2 * x) + (3 * (-y))

-u' = -(u')

u * v' = u * (v')

u' * v = (u') * v

-n ^ 3 = -(n ^ 3)

a > 0 || b < 0 ? c + d : e - f = (a > 0 || b < 0) ? (c + d) : (e - f)

a ? b : c ? d : e = a ? b : (c ? d : e)
